# ApiPortal_Exercise (Test for 23People)
apiservice with nodejs, docker and gcp

## Development stack
- NodeJS / Express for BackEnd
- Apigee for Authentication

## FrontEnd
__Development__
```
cd app.js
npm install
npm start```

__Build__
```
npm run-script build
```

## Deployment on GCP

```
gcloud app deploy
gcloud app browse
```
